# PTH XT-IDE Boards

This directory contains through-hole (PTH) versions of the Just Another XT-IDE board designs.

The goal of these revisions is to provide:

- Easy hand assembly.
- Component alignment.
- Switches access.
- Fix previous errors.

## Version Overview

| Version | Status | Notes |
|----------|--------|-------|
| v0.4 | Prototype | Improved routing, mechanical adjustments and silkscreen |

## Design Philosophy

The PTH variants are intended for:

- Hobbyist assembly.
- Easier troubleshooting.
- Retro hardware experimentation.
- Low-cost PCB manufacturing.

## Notes

Each version directory contains:

- Gerber files.
- Bill of materials (BOM).
- Configuration documentation.
