## Compatibility

Tested with:

- IBM XT compatible systems
- CompactFlash IDE adapters
- Standard IDE hard drives
- XTIDE Universal BIOS

## Notes

This board is intended for experimental and hobbyist retrocomputing use.

Some configurations may require BIOS changes depending on the target system.
